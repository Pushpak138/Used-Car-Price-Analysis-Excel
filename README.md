# Used-Car-Price-Analysis-Excel 🚗 
# 📌 Project Overview
The primary objective of this project is to analyze trends in the used car market and understand the factors affecting resale value. The dataset consists of over 7,400+ vehicle records, processed and visualized using advanced Excel techniques.

# 📊 Key Insights from Analysis
Through detailed Pivot Tables and interactive Dashboards, the following major findings were identified:

* Market Dominance: Maruti (with ₹127Cr+ total sales value) and Hyundai are the leading brands in the dataset.

* Regional Trends: The highest inventory concentration is in Maharashtra (2,118 cars) and Delhi (1,295 cars).

* Popular Models: Maruti Swift emerged as the high-demand model with significantly higher annual views compared to others.

* Fuel & Body Type: The market is dominated by Petrol engines (4,659 units) and Hatchbacks (4,359 units).

* Depreciation Study: Conducted a detailed analysis of price depreciation based on car age and mileage (kilometers run).

# 🛠️ Technical Skills Used
* Data Cleaning: Handled missing values, date formatting (ad_created_on), and categorical data management.

* Data Processing: Engineered custom metrics such as depreciation_per_km and Car_Age.

* Pivot Tables & Charts: Summarized complex data points like fuel type, transmission, and state-wise distribution.

* Interactive Dashboard: Developed a user-friendly interface using Slicers and Timelines for dynamic data exploration.

# 📁 Repository Structure
* Used car Data.xlsx: The primary Excel workbook containing raw data, cleaning steps, and the final dashboard.

* Screenshots/: Visual gallery showcasing the dashboard and key analytical charts.

# 💡 How to Use
Download the Used car Data.xlsx file from this repository.

1) Open the file in Microsoft Excel.

2) Navigate to the 'Dashboard' sheet.

3) Use the interactive slicers to filter data by car make, model, or registered state.
